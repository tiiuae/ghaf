### Development Workflow

1. **Explore**: Understand the codebase structure
2. **Plan**: Identify which layers to modify
3. **Implement**: Write modules following Ghaf patterns
4. **Test**: Build and run in VM
5. **Validate**: Run `nix fmt` and `nix flake check`
6. **Document**: Update relevant documentation

### Key Concepts to Understand

Before diving into development:

1. **[NixOS Module System](https://nixos.wiki/wiki/NixOS_modules)** - Foundation of all Ghaf configuration
2. **[5-Layer Architecture](/ghaf/dev/architecture/module-layers)** - How configuration composes
3. **[VM Architecture](/ghaf/dev/architecture/vm-architecture)** - How VMs are structured
4. **[specialArgs](/ghaf/dev/guides/creating-modules#specialargs)** - How dependencies are passed
